# Welcome to Mock API

A RESTful API with fictional data for testing and development. Use it to test, prototype and develop your application without connecting to a real API.

## Base URL

```
https://mock-api-ten-eosin.vercel.app/api
```

## Quick Start

All endpoints are prefixed with `/api` and require no authentication:

```
GET /api/users       # List users (Pagination supported)
GET /api/products    # List products (Pagination supported)
```
